# v1.0.0 更新说明

v1.0.0 是团绒喵记本的首个正式线上版本，建立了校园猫协地图、任务、猫咪资料与成员管理的完整基础。

## 用户侧功能

- 支持邀请制成员登录、验证码校验、首次改密和个人资料初始化。
- 提供校园地图点位、筛选与搜索、腾讯 POI 关联、步行路线导航和点位详情。
- 支持喂食任务发布、编辑、状态流转、周期任务执行和完成照片记录。
- 提供猫咪库列表、统计、筛选和详情入口。
- 提供个人中心，以及管理员人员、任务、物资点和地标点管理入口。
- 图片统一上传至对象存储，并在任务和业务详情中稳定回显。

## 后端与数据

- 建立 FastAPI、Pydantic、SQLAlchemy、Alembic、PostgreSQL 与 PostGIS 基础设施。
- 统一 API 响应、错误处理和 trace ID，并接入鉴权与角色权限控制。
- 完成地图、任务、猫咪库、个人资料、管理员和文件上传等首批业务表与接口。
