# v1.1.0 更新说明

v1.1.0 是喵记信息架构、药品管理和生产 HTTPS 上线版本。

## 用户侧更新

- 将原任务入口重构为喵记书架，集中提供任务、物资、地标和药品入口。
- 新增任务、物资点和校园地标列表，优化详情页、地图卡片和图片预览体验。
- 新增药品分类、药品主档、成员持有库存、库存流水和用药申请流程。
- 支持管理员维护药品分类、审核用药申请，以及归档或删除药品。
- 修复周期任务归档、取消与子任务状态归一化问题。
- 修复管理员任务、物资和地标选点时初始位置与重置位置不正确的问题。

## 后端与数据

- 新增药品管理接口、模型和数据库迁移 20260706_0012。
- 生产后端切换到公开 CA HTTPS 域名，HTTP 自动跳转 HTTPS。
- 前端生产请求统一使用 HTTPS API，部署脚本补充环境、CORS、迁移和健康检查门禁。

## 发布与兼容性

- 小程序版本：1.1.0，版本代码：110。
- 发布时后端 135 项测试、Ruff、前端 265 项测试、类型检查、微信小程序构建和部署契约均通过。
- 本 Release 于 2026-07-17 根据原有 annotated tag 与发布记录补建；代码仍指向 2026-07-09 创建的原始 v1.1.0 tag。
