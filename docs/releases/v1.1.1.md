# v1.1.1 更新说明

v1.1.1 是微信 OpenID 登录安全增强与地图、任务、资料交互稳定性版本。

## 用户侧更新

- 新增微信小程序 OpenID 首次绑定和后续自动登录，密码登录继续作为兼容兜底。
- 支持成员自行解除微信绑定，以及管理员清除成员绑定后重新登录绑定。
- 完善人员账号操作、头像对象存储上传和资料离页未保存提醒。
- 优化任务状态派生、任务记录展示和喂食任务地图标记。
- 优化地图抽屉拖拽、点位气泡、点位聚焦动画与选中点自动展开。
- 更新加载页视觉，并改进多图预览和返回交互。

## 后端与数据

- 新增 POST /api/v1/auth/wechat/login 和管理员微信解绑接口。
- 用户表新增微信 OpenID 绑定与登录时间字段，并增加非空唯一索引。
- 新增数据库迁移 20260709_0013。
- 生产鉴权保持 optional 兼容模式，旧版小程序和密码登录仍可继续使用。

## 发布与兼容性

- 小程序版本：1.1.1，版本代码：111。
- 后端已部署；历史发布记录显示当时微信小程序 1.1.1 构建已完成，但尚未上传为正式版本。
- 发布时后端 167 项测试、Ruff、前端 307 项测试、类型检查、微信小程序构建和部署契约均通过。
