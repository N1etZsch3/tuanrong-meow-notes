# v1.1.2 更新说明

v1.1.2 是图片内容安全、头像审核和资料编辑稳定性热修版本。

## 用户侧更新

- 头像上传接入微信异步内容安全审核，补充待审核、通过、拒绝和失败状态提示。
- 任务、地标、物资和药品等普通业务图片上传后可立即使用，不再被头像审核流程错误阻塞。
- 修复资料页和管理员成员编辑页返回时的重复确认、返回拦截和脏表单判断问题。
- 修复头像审核通过后仍回退到默认头像、资源地址被小程序解析到本地页面以及失败提示重复出现的问题。
- 修复管理员保存成员资料时，头像审核字段导致操作日志写入失败的问题。

## 后端与数据

- 新增微信内容安全回调接口、审核结果幂等处理和图片安全状态字段。
- 支持微信 XML 嵌套回调结果，审核通过后可自动应用头像。
- 历史普通业务图片按兼容状态继续使用，无需批量重传或重新审核。
- 新增数据库迁移 `20260712_0014`。

## 发布与兼容性

- 小程序版本：`1.1.2`，版本代码：`112`。
- 后端继续兼容旧版小程序；图片审核配置、回调 Token 和生产健康检查由部署脚本强制校验。
- 已完成后端、前端、迁移、生产部署和真机回归验证。
