# v1.1.3 更新说明

v1.1.3 是任务、图片预览、图片排序和成员自动编号热修版本。

## 用户侧更新

- 修复任务动态时间直接显示 UTC 的问题，统一按中国标准时间（UTC+8）展示。
- 大图查看改用微信小程序原生图片预览，支持稳定的多图切换、双指缩放、放大后平移和长按菜单；删除自定义大图手势组件。
- 修复周期任务编辑状态作用范围错误：父任务入口修改父任务，子任务入口只修改对应执行日期的子任务。
- 任务点、物资点、药品和地标图片支持拖拽排序，保存后第一张图片作为列表封面。
- 修复手动指定高位喵喵号会推高自动编号的问题；自动号码改用独立序列，手动号码不再破坏连续编号。

## 后端与数据

- 新增管理员子任务状态接口，并保持现有父任务生命周期派生规则。
- 药品详情和编辑支持有序多图数据。
- 新增 PostgreSQL 自动喵喵号 sequence，迁移为 `20260716_0015`。
- 生产测试账号已按授权清理，自动编号已恢复并通过实际新增成员测试。

## 发布与兼容性

- 小程序版本：`1.1.3`，版本代码：`113`。
- 后端变更为增量兼容，不删除现有字段，不改变旧版小程序已有接口行为。
- 已完成后端 195 项、前端 317 项自动测试、类型检查、微信小程序构建、生产健康检查和全部真机功能验收。
